# Odd Even (Java)

A program to check whether a given number is Odd or Even.

## How to Run:
1. Open a terminal and navigate to this folder:
   ```bash
   cd java/basic/odd-even
3. Compile the program:
   ```bash
   javac OddEven.java
5. Run the program:
   ```bash
   java OddEven
7. Output:
   ```bash
   Enter a number: 6
   6 is an even number.
